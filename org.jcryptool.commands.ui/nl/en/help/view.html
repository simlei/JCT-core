Console

The JCrypTool Console enables you to directly enter commands inside JCrypTool. To start the view either click the console icon in the toolbar or use the usual way via the Window -> Show view dialog.

The console is by default shown at the bottom of the JCrypTool window. You can directly start typing commands, just like in the Windows command prompt or a terminal on Linux or macOSX.

Usage

The whole console is text oriented and only offers a minimal toolbar. The only action available here is to delete the views content. All other commands have to be entered manually.

Use the command help for general assistance. Without any other parameter you get an overview about all available commands. This includes algorithms. The amount of available commands depends on the installed plug-ins in your environment (plug-ins can provide their own extensions to the Console view).

A more detailed description for one of the available commands can be shown by entering help commandname, e.g. help caesar. This shows you the syntax of all available parameters/ options of the command. The commands and their possible parameters/ options do differ between the crypto plug-ins. Some commands may not be available everywhere, or they result into different operations. So simply use the command help via help commandname to get all this information.

Command Overview

Command Description
help [Command] Without any following command or parameter (entering help only) all available commands will be listed (including the algorithms). With another command or parameter (e.g. help caesar) the syntax of the given command with its parameters and options will be shown.

Limitations

Not every crypto plug-in can be used via the console. Only algorithm plug-ins (from classic to modern) in principal do support the console. Analysis, Games and Visualizations are not supported. In order for an algorithm to be used via the console, the plug-in must provide certain extensions. This requires the plug-in developer to extend a so called Extension Point with his plug-in. However this extension is optional, so even if a crypto plug-in is available in the menu or the Crypto Explorer view it is not necessarily available in the Console view too.
