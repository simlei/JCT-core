Konsole

Die JCrypTool-Konsole ermöglicht die direkte Eingabe von kryptografischen Befehlen innerhalb JCrypTools. Klicken Sie zum Aufruf der Sicht auf das Konsole Icon in der Toolbar oder verwenden Sie den üblichen Weg über das Menü Fenster -> Sicht anzeigen.

Die Konsole wird standardmäßig am unteren Rand des JCrypTool Fensters angezeigt. Sie können direkt mit der Eingabe von Befehlen beginnen, in der Art der Windows Eingabeaufforderung oder einem Terminal unter Linux bzw. macOS.

Bedienung

Die Konsole ist textorientiert und verfügt nur über eine minimale Toolbar. Hier können Sie ausschließlich den Inhalt der aktiven Konsole löschen. Alle weiteren Befehle können nur über Kommandos direkt eingegeben werden.

Das Kommando help hilft Ihnen hier immer weiter. Ohne weiteren Parameter erhalten Sie damit einen Überblick über alle verfügbaren Befehle. Das schließt Algorithmen mit ein. Die Anzahl der verfügbaren Befehle hängt von den bei Ihnen installierten Plug-ins ab (Plug-ins können ihre eigenen Kommandos beisteuern und die Konsole erweitern).

Umfangreichere Hilfe zu einem der verfügbaren Kommandos erhalten Sie mit help Kommandoname, z.b. help caesar. Hiermit wird Ihnen die Syntax samt aller möglichen Parameter/ Optionen des Befehls angezeigt. Die Befehle und mögliche Parameter/ Optionen unterscheiden sich in den einzelnen Krypto Plug-ins. Manche Kommandos sind nicht überall verfügbar, oder sie lösen verschiedene Aktionen aus. Verwenden Sie daher Kommandohilfe über help Kommandoname um diese Informationen anzuzeigen.

Befehlsübersicht

Beachten Sie, dass die Befehle immer auf Englisch eingegeben werden müssen, auch wenn die Beschreibungen möglicherweise auf Deutsch angezeigt werden.

Befehl Beschreibung
help [Kommando] Ohne weiteres Kommando oder Parameter (d.h. nur Eingabe von help) werden alle vorhandenen Befehle aufgelistet (auch die vorhandenen Algorithmen). Mit weiterem Kommando bzw. Parameter (z.B. help caesar) wird die Syntax samt der möglichen Parameter und Optionen für dieses spezielle Kommando angezeigt.

Einschränkungen

Nicht jedes Krypto Plug-in kann über die Konsole aufgerufen werden. Nur Algorithmen Plug-ins (von klassisch bis modern) unterstützen prinzipiell die Konsole. Analysen, Spiele und Visualisierungen werden nicht unterstützt. Damit der Algorithmus eines Plug-ins über die Konsole ausgeführt werden können, müssen bestimmte Erweiterungen vom Plug-in zur Verfügung gestellt werden. D.h. der Plug-in Entwickler muss einen sogenannten Erweiterungspunkt (Extension Point) mit seinem Plug-in erweitern. Diese Erweiterung ist optional, d.h. auch wenn ein Krypto Plug-in in den Menüs oder der Krypto-Explorer Sicht auftaucht ist die Erweiterung der Konsole davon unabhängig.
